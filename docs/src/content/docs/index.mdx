## Quick Usage
<Tabs>
	<TabItem label="push">
		Push your environment variables to git repository

		```sh
		npx envshh push
		```
		<Image src={pushGif} alt="tutorial" width="1200"/>
	</TabItem>
	<TabItem label="pull">
		Pull your environment variables from the git repository

		```sh
		npx envshh pull
		```
		<Image src={pullGif} alt="tutorial" width="1200"/>
	</TabItem>
	<TabItem label="generate">
		Generate a `.env.example` file

		```sh
		npx envshh generate
		```
		<Image src={generateGif} alt="tutorial" width="1200"/>
	</TabItem>
	<TabItem label="clone">
		git clone and pull your environment variables from the git repository at the same time

		```sh
		npx envshh clone [git-repo-url]
		```
		<Image src={cloneGif} alt="tutorial" width="1200"/>
	</TabItem>
</Tabs>
