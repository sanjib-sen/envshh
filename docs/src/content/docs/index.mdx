## Quick Demo

<Tabs>
	<TabItem label="push">
		Push your environment variables to a private git repository
  
    	```sh
    	npx envshh push
    	```
      <video width="1478" height="772" controls>
        <source src={pushTutorial} type="video/webm"/>
      </video>

    Learn more about the [Push Command](/envshh/commands/01-push)

    </TabItem>
    <TabItem label="pull">
    	Pull your environment variables from the git repository

    	```sh
    	npx envshh pull
    	```
      <video width="1478" height="772" controls>
        <source src={pullTutorial} type="video/webm"/>
      </video>
    Learn more about the [Pull Command](/envshh/commands/02-push)

    </TabItem>
    <TabItem label="generate">
    	Generate a `.env.example` file

    	```sh
    	npx envshh generate
    	```
      <video width="1478" height="772" controls>
        <source src={generateTutorial} type="video/webm"/>
      </video>
    Learn more about the [Generate Command](/envshh/commands/03-generate)

    </TabItem>
    <TabItem label="pipe">
    	Say goodbye to dotenv. Pipe your environment variables to any command

    	```sh
    	npx envshh pipe [command]
    	```
      <video width="1478" height="772" controls>
        <source src={pipeTutorial} type="video/webm"/>
      </video>

    Learn more about the [Pipe Command](/envshh/commands/04-pipe)

    </TabItem>
    <TabItem label="clone">
    	git clone and pull your environment variables from the git repository at the same time

    	```sh
    	npx envshh clone [git-repo-url]
    	```
  
      <video width="1478" height="772" controls>
        <source src={cloneTutorial} type="video/webm"/>
      </video>

      Learn more about the [Clone Command](/envshh/commands/05-clone)

    </TabItem>
    <TabItem label="branch">
      Use push or pull for different versions of .env such as staging, production, etc.

    	```sh
    	npx envshh push -b [branchName]
    	```
      And

      ```sh
      npx envshh pull -b [branchName]
      ```

      <video width="1478" height="772" controls>
        <source src={branchTutorial} type="video/webm"/>
      </video>

      Learn more about the [Branching](/envshh/core-concepts/02-branch)

    </TabItem>

</Tabs>
